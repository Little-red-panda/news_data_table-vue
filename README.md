# Первая работа на Vue.js

# Небольшой шаблон с выводом новостей из открытого API в виде таблицы. Доступна сортировка по дате и источнику и фильтрация по поисковой строке. Так же можно добавлять свои новости.

# Запустить проект
```
npm run serve
```
